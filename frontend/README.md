# Frontend (Next.js App Router)

- Este diretório contém a aplicação frontend modular usando Next.js App Router.
- Rotas reais sob /app (layout.tsx já existente) e novas páginas criadas (dashboard, agenda, configuracoes, relatorios, clientes).
- Para desenvolvimento: `cd simple-erp && npm install` e `npm run dev:frontend`.
- O frontend consome a API Express existente (health, echo, self-heal).
